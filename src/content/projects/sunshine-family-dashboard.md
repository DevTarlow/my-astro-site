The Sunshine Family Dashboard is a lightweight, local-first web application designed to be a simple but effective smart home dashboard for your household. Packaged in a single Docker container and built with Next.js and SQLite, it allows your family to seamlessly manage weekly dinner menus, shared grocery lists, daily to-dos, and sticky notes. It also features a live local weather widget and a rotating carousel of your favorite family photos. Built with privacy in mind, Family Dashboard requires no cloud databases or subscriptions and it runs entirely on your own local network, ensuring your family's data stays fast, secure, and 100% yours.

## Features

- Local Member login / profile with per‑member theme and background
- Real‑time presence & activity feed
- Weather widget (OpenWeatherMap)
- Vibe‑of‑the‑Day photo
- Cannon Beach live cam
- Photo carousel with favorites and upload
- Weekly dinner planner with AI suggestion and comments
- Fitness tracker (weekly log)
- Dining‑out tracker (monthly)
- Todo list
- Grocery list
- Notes with comment threads
- Meal‑prep fridge (Gemini‑generated emoji)
- Shared links board
- Achievements system
- Unread badges for new items/comments
- Auto‑refresh every 30 s
- Dark / light mode per member

## Tech Stack

Built with modern web technologies for optimal performance and developer experience.
