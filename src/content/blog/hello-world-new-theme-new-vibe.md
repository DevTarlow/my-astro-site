I thought to my self:

> "Isn't this a space where I should express creativity, push myself, and learn? Well then why don't I create my own Astro site theme for the website."

I really liked the space theme I was using previously but it was just too distracting while trying to focus on reading the content.

So I started building. One prompt after the next, the vision became a reality.

All within 2 days.

## Built With

- **Astro 6**
- **Tailwind CSS 3**
- **TypeScript**

## Theme

Minimal Modern Hipster — Digital Paper (`#F9F9F8`) surfaces, Terracotta (`#C05746`) accents, Playfair Display for headings, Inter for body text.


## How did I build it?

Vibe coding.

No, but seriously, I just talked to AI to refine the idea, get guidance, construct the plan, and then write the code.

Today, we are more the orchestrators than the drivers. Standing behind the scenes delegating and watching AI agents to take the wheel.

My setup and flow for this one were simple:
1. Asked [Gemini](https://gemini.google.com/app) using my custom OpenCode notebook (it is designed with a guided system prompt to understand my mission for projects) that I was wanting to build my own astro theme for my website. With a few design details sprinkled in.
2. Gemini then helped me refine the vision and then gave me the Master Prompt to give the AI agent running in Opencode (lately OpenCode Go has been amazing).
3. Pasted the refined master prompt in [OpenCode](https://opencode.ai/) and we were off to the races! After that it was just working with the codebase in OpenCode desktop to build the features. 
4. Then do the same thing we did using Gemini but in OpenCode and with our codebase, refine and optimize by asking questions about the project so it can point out what is missing before I push it live.
5. Pushed the site live to github pages, and now we have the site you are seeing today.
